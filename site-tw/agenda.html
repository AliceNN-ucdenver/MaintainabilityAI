POP/POP/POP Agentic Engineering Bootcamp

From vibe coding to production‑grade agentic workflows—security‑first, maintainability always.

When

Full‑day (7.5 hours) or two half‑days

Where

VS Code + GitHub + GH Actions + Snyk + CodeQL

Who

Junior devs & senior engineers; security champions; tech leads

Agenda

  1. 1) The Spectrum: Vibe → Assisted → Agentic

    70% problem & the Human 30%. Golden Rules of Vibe Coding. Tooling tour.

  2. 2) Security‑First Prompting

    OWASP‑aligned prompt packs, CoT for threat thinking, reusable patterns.

  3. 3) Hands‑on: A03 Injection

    Remediate an insecure TS module using Claude/Copilot; make tests pass; CodeQL & Snyk gates.

  4. 4) Agentic Workflows in VS Code

    Plan → Execute → Verify → Report with PR checklist & governance.

  5. 5) Multi‑Agent QA & Hardening

    Reviewer agent, test writer, security validator; keeping drift out.

  6. 6) Measuring & Improving

    Security SLIs/SLOs, autofix acceptance rate, MTTR/MTTD; retros.

What to install

  • VS Code latest with GitHub Copilot / Claude Code
  • Node.js 20+
  • GitHub account with Actions enabled
  • Snyk account & token (optional but recommended)
